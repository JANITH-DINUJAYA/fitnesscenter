Fitness Center Webpage built with React and Vite.

View the live project here:https://fitnesscenter-xi.vercel.app/

Figma Design Link:https://www.figma.com/design/aCFDRC1OLGXfdcup0aoP2h/Untitled?node-id=1-22&t=2RfEQYL0v2i2e6vl-1

Tech Stack: React.js
Deployment:Vercel
Styling:Bluestrap CSS

Setup Instructions

1.Clone The Repositary
2.Install Dependencies(npm install,npm install lucide-react,npm install react-router-hash-link,npm install react-router-dom)
3.Run the development server(npm run dev)
4.Build The Product(npm run build)
